# Hi :)

I'm a Mechanical Engineering student at TUM, focused on building hardware-based systems.

This repository is mainly used to document my projects and experiments.

Most of the things I build are not originally designed to be published. I tend to lose track of details over time, so this space helps me structure and document my work more clearly for myself.

At the same time, it bothers me how much effort gets wasted because people repeatedly solve the same problems — for example when everyone has to set up their own configs or workflows from scratch.

This repository is my attempt to reduce that by sharing selected projects, guides and workflow improvements that others can reuse or build on.

## Focus
- Embedded systems (ESP32, Arduino)
- Hardware prototyping
- Robotics and control systems
- 3D design and printing

## Projects & Guides
Content will be added step by step.

The goal is to cover:
- Selected projects that solve specific practical problems
- Guides and setups to simplify workflows
- Configurations and tools that improve efficiency

Some examples of ongoing work:
- Line-following robot with BLE control  
- 400W portable speaker with custom control panel and DSP
